# ShareCode

A collaborative code sharing application that allows developers to create and share code snippets in real-time.

Built with:

- Next.js
- Socket.IO

### Getting Started

```bash
# Clone the repository
git clone https://github.com/ismailgunayy/sharecode

# Run the server:
cd sharecode/server
yarn
yarn dev

# Run the client:
cd sharecode/client
yarn install
yarn dev
```
